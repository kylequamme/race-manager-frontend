# Race-Manager-Frontend
This is a next js frontend for the Race-Manager application. It is imported as a submodule to the Race-Manager repo.  

Race-Manager keeps track of points earned during NR2003 league events

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your favorite browser

See the Race-Manager project for more information
